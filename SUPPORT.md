# Support

## Getting Help

Before opening an issue, please:

1. Read the project's README and any available documentation
2. Search [existing issues](../../issues) to see if your question has been answered
3. Search [Stack Overflow](https://stackoverflow.com) for related topics

If you still need help, [open an issue](../../issues/new) with a clear description of your problem and what you've already tried.

## What This Isn't

These are open source projects maintained in my spare time. There are no guarantees of support or response times. Please do not reach out via social media or personal email for project support — use GitHub Issues so others can benefit from the discussion.
